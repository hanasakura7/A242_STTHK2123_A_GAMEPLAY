# A242_STTHK2123_A_GAMEPLAY
Groupb-based Game enhancement Project for A242 Session, Subject Interaction Systems and Tools

YouTube video (original gameplay + source code): https://www.youtube.com/watch?v=1lneuIhrfh4

Initial code (GitHub): https://github.com/clickclackcode/python-side-scroller

For this project, we enhanced the features of the gameplay by doing the following:

- Added Main Menu when the game runs, with buttons of options allowing player to Start of Quit the Game

- Added Settings feature for players to have the option to switch the Audio and Sound Effects, on and off

- Added Tutorial Page

- Added Sound Effect, Gameplay Music and Game Over Music

- Added collision effect

- Added another obstacle, clouds. If the player clashes with it, 1 life will be deducted

- Added power-up effect

- Added level up feature; level's difficulty will be increased when the game reaches level 20 and will continue increasing every 20 levels

- Added Game Over Screen, displaying previous score, current score, and button options to either play again, return to main menu, or quit
